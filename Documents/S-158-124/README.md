# Product-specific checks for S-124 (Navigational Warnings)

Documents and materials pertaining to the S-124 (Navigational Warnings) Product Specification.

This directory contains "product-specific" checks prepared in the S-100 Validation Checks sub-group from the list supplied by the S-124 Project Team. Now handed off to the project team, location for subsequent development is up to the project team.

2024-12-11: Spreadsheet file with list of checks added in 0.2.0 folder for completeness. No substantive changes to the 0.1.0 list of checks. Editorial changes in metadata worksheet and population of Check ID column with 124_XXXX.

2024-12-04: Edition 0.2.0 cover document, including current thinking in S-100 validation sub-group concerning versioning.

The documents in this directory and its sub-directories are development versions. The formally published specification will be made available from the IHO Geospatial Information Registry.
